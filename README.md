# 🚀 ProspectMiner-AI

ProspectMiner-AI is a MERN stack-based application designed to automate lead generation through web scraping and provide intelligent insights via an interactive dashboard.

---

## 📌 Features

### 🔍 Lead Generation
- Generate leads by entering search text (e.g., Restaurants, Hospitals)
- Location-based scraping for targeted results
- Dynamic data extraction using web scraping

### 📊 Dashboard
- View all generated leads in a structured table
- Lead scoring system: High, Medium, Low
- Displays total leads and distribution

### 🎯 Advanced Filtering
- Filter leads by:
  - Category
  - Location
  - Lead Score
- Search functionality for quick access

### 🕒 Scrape History
- Track all scraping jobs
- View job status and timestamps
- Actions:
  - View leads
  - Export to Excel
  - Delete jobs

### 📥 Export Data
- Download leads in Excel format for business use

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other:** Web Scraping, REST APIs
